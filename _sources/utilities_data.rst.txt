run_data
==========================

.. automodule:: utilities.run_data
   :members:
   :exclude-members: 
   :undoc-members:
   :show-inheritance:

To run a data job:

.. code-block:: bash

   run_data.py --config templates/config.yaml --sources examples/sources.yaml

Alternatively, you can use the executable entrypoint:

.. code-block:: bash
   
   bahamas-data --help
